# ai-art-detector

Website that detects if artwork is AI-generated. Uses Angular, Django, Keras and SQLite.

To-do: screenshots.

# About

AI Art Detector is a single-page web app that lets users upload images and determines if they are AI-generated using a binary image classifier CNN.

The frontend uses Angular and the backend uses Django, Keras and SQLite.

## Data

The CNN was trained on a subset of a combination of datasets:

- LAION-Aesthetics V2: A dataset of 'aesthetic' images, used by many to train text-to-image models including Stable Diffusion v1.

- Open Prompts: A dataset of images generated by Stable Diffusion v1.3.

In the future, I would like to expand the training data and include images from other popular text-to-image models such as DALL-E and Midjourney.
